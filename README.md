# Project Vault

Live project data maintained by the project-manager agent.
Updated automatically from team Slack check-ins.

## Quick links

- [Current Status](Dashboards/Now.md)
- [Blockers](Dashboards/Blocked.md)
- [This Week](Dashboards/This-Week.md)

## What's here

| Folder | Contents |
|--------|----------|
| Checkins/ | Daily check-in records from team members |
| Goals/ | Active goals with owners and KPIs |
| Milestones/ | Milestone tracking with status and evidence |
| Reports/Weekly/ | Weekly summary reports |
| Dashboards/ | Auto-generated status views |

## How data gets here

Team members post check-ins in Slack. The project-manager agent
parses replies, stores structured data here, and generates
dashboards and reports automatically.

This repo is write-only by the agent. Do not edit files directly.
